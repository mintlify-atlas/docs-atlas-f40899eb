## Overview

Inmobiliaria Web is a comprehensive web application designed for managing real estate properties (products) and their categories. Built with modern web technologies including HTML5, CSS3, JavaScript, and Bootstrap 5, it provides an intuitive interface for property management operations.

## Technology Stack

The application is built using:

- **Frontend Framework**: Bootstrap 5.3.2 for responsive UI components
- **JavaScript**: Vanilla JavaScript with Fetch API for backend communication
- **Icons**: Bootstrap Icons for consistent iconography
- **Architecture**: Single-page application with iframe-based navigation

## Use Cases

Inmobiliaria Web is ideal for:

- **Real Estate Agencies**: Manage property listings and categorize inventory
- **Property Managers**: Track properties across different categories and price ranges
- **Inventory Management**: Monitor real estate inventory with real-time analytics
- **Small to Medium Businesses**: Lightweight solution without complex dependencies

## Backend Integration

The application requires a backend API server running on `http://localhost:8080`. The backend provides RESTful endpoints for:

- Product (property) management
- Category management
- Statistical summaries and analytics

See the [API Integration](/development/api-integration) guide for details on connecting to your backend.
