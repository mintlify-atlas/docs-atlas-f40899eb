## Overview

The Dashboard provides a comprehensive view of your product inventory with real-time statistics, category breakdowns, and key metrics. Data automatically refreshes every 30 seconds to ensure you always have current information.

<Note>
  The dashboard fetches data from the `/api/producto/resumen` endpoint and automatically updates every 30 seconds to provide real-time insights.
</Note>

## Main Statistics

### Fetching Summary Data

The dashboard retrieves comprehensive statistics from the backend:

```javascript
async function obtenerResumen() {
  try {
    const response = await fetch("http://localhost:8080/api/producto/resumen");
    
    if (!response.ok) {
      throw new Error("Error al obtener el resumen");
    }
    
    const data = await response.json();
    renderizarResumen(data);
  } catch (error) {
    console.error("Error al cargar el resumen:", error);
    mostrarError();
  }
}
```

### Auto-Refresh

The dashboard automatically updates every 30 seconds:

```javascript
// Initial load
obtenerResumen();

// Update statistics every 30 seconds
setInterval(obtenerResumen, 30000);
```

<Tip>
  The 30-second refresh interval ensures data stays current without requiring manual page refreshes. This is particularly useful when monitoring inventory changes in real-time.
</Tip>

## Rendering Statistics

The `renderizarResumen()` function populates all dashboard elements:

```javascript
function renderizarResumen(data) {
  // Main statistics
  document.getElementById("resTotalProductos").textContent =
    data.totalProductos || 0;
  document.getElementById("resTotalCategorias").textContent =
    data.totalCategorias || 0;
  document.getElementById("resInventario").textContent = formatearMoneda(
    data.valorInventario || 0
  );
  document.getElementById("resPromedio").textContent = formatearMoneda(
    data.precioPromedio || 0
  );

  // Inventory by category status
  document.getElementById("resInventarioActivo").textContent = formatearMoneda(
    data.valorInventarioCategoriasActivas || 0
  );
  document.getElementById("resInventarioInactivo").textContent =
    formatearMoneda(data.valorInventarioCategoriasInactivas || 0);

  // Most expensive product
  if (data.productoMasCaro) {
    document.getElementById("resMasCaroNombre").textContent =
      data.productoMasCaro.nombre;
    document.getElementById("resMasCaroPrecio").textContent = formatearMoneda(
      data.productoMasCaro.precio
    );
  }

  // Cheapest product
  if (data.productoMasBarato) {
    document.getElementById("resMasBaratoNombre").textContent =
      data.productoMasBarato.nombre;
    document.getElementById("resMasBaratoPrecio").textContent = formatearMoneda(
      data.productoMasBarato.precio
    );
  }

  // Products by category
  renderizarProductosPorCategoria(data.productosPorCategoria || []);
}
```

## Currency Formatting

All monetary values are formatted as Colombian Pesos:

```javascript
function formatearMoneda(valor) {
  return new Intl.NumberFormat("es-CO", {
    style: "currency",
    currency: "COP",
    minimumFractionDigits: 0,
    maximumFractionDigits: 0,
  }).format(valor);
}
```

<Note>
  The `Intl.NumberFormat` API formats currency according to Colombian standards, displaying values like "$1.500.000" for 1,500,000 pesos.
</Note>

## Product Distribution by Category

The dashboard visualizes how products are distributed across categories with progress bars and percentages.

### Rendering Distribution

```javascript
function renderizarProductosPorCategoria(productos) {
  const contenedor = document.getElementById("resPorCategoria");

  if (!productos || productos.length === 0) {
    contenedor.innerHTML = '<p class="text-muted">No hay datos disponibles</p>';
    return;
  }

  // Calculate total for percentages
  const total = productos.reduce((sum, item) => sum + item.cantidad, 0);

  contenedor.innerHTML = productos
    .map((item) => {
      const porcentaje =
        total > 0 ? ((item.cantidad / total) * 100).toFixed(1) : 0;

      return `
        <div class="col-md-4">
          <div class="card border-0 shadow-sm">
            <div class="card-body">
              <div class="d-flex justify-content-between align-items-center mb-2">
                <h6 class="mb-0">${item.categoria}</h6>
                <span class="badge bg-primary">${item.cantidad}</span>
              </div>
              <div class="progress" style="height: 20px;">
                <div class="progress-bar bg-primary" role="progressbar" 
                     style="width: ${porcentaje}%;" 
                     aria-valuenow="${porcentaje}" 
                     aria-valuemin="0" 
                     aria-valuemax="100">
                  ${porcentaje}%
                </div>
              </div>
            </div>
          </div>
        </div>
      `;
    })
    .join("");
}
```

## Error Handling

When the API fails to load, an error alert is displayed:

```javascript
function mostrarError() {
  const contenedor = document.querySelector("main .container");

  const alerta = document.createElement("div");
  alerta.className = "alert alert-danger alert-dismissible fade show";
  alerta.innerHTML = `
    <i class="bi bi-exclamation-triangle-fill"></i>
    <strong>Error:</strong> No se pudo cargar el resumen del inventario.
    <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
  `;

  contenedor.insertBefore(alerta, contenedor.firstChild);
}
```

<Warning>
  If the API is unavailable or returns an error, the dashboard displays a dismissible alert at the top of the page. The alert includes an icon, error message, and close button.
</Warning>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Dashboard shows all zeros">
    This typically means:
    - The API is not reachable
    - No products exist in the database
    - The backend service is down
    
    Check the browser console for error messages and verify the API endpoint is accessible.
  </Accordion>
  
  <Accordion title="Statistics not updating">
    If the dashboard loads initially but doesn't refresh:
    - Check browser console for interval errors
    - Verify the API is responding to subsequent requests
    - Check for JavaScript errors that may have stopped execution
  </Accordion>
  
  <Accordion title="Currency not formatting correctly">
    Ensure:
    - The browser supports `Intl.NumberFormat`
    - The locale string "es-CO" is valid
    - Values are numeric (not strings)
  </Accordion>
  
  <Accordion title="Distribution section empty">
    This occurs when:
    - No products exist
    - The `productosPorCategoria` array is empty
    - Categories have no products assigned
  </Accordion>
</AccordionGroup>
